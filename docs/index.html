Get Started

1

Install via Homebrew

The easiest way to install Tudu is using Homebrew.

brew tap AslamSDM/tudu brew install --cask tudu
2

Launch & Enable Widgets

Open Tudu from your Applications folder. To add widgets:

  1. Open Notification Center
  2. Click "Edit Widgets"
  3. Search for "Tudu"
  4. Drag your preferred widget size to the desktop
