# Tudu Landing Page

This folder contains the static landing page for the Tudu app that can be hosted on GitHub Pages.

## Files

- `index.html` - Main landing page
- `styles.css` - Custom styles (if created separately)
- `images/` - Folder for screenshots and other images (if needed)

## GitHub Pages Setup

To host this landing page on GitHub Pages:

1. Go to your repository settings on GitHub
2. Scroll down to the "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/docs" folder
5. Click "Save"
6. Your landing page will be available at: https://[username].github.io/tudu/

The page will automatically update when you push changes to the main branch.
